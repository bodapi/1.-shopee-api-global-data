## 🛠️ 快速集成指南 (Quick Integration Guide)

我们的 Shopee API 采用专业的**异步任务工作流**，在绕过高级反爬虫保护的同时处理高并发请求。

### 🔄 3 步集成流程：

1. **提交任务 (Submit Task)** 📤  
   调用 [提交产品详情](https://bodapi.com/en/apis/shopee-api/submit-product-detail) 接口创建抓取任务，系统将返回 `batch_id`。

2. **轮询状态 (Poll Status)** 🔍  
   使用 [查询产品详情](https://bodapi.com/en/apis/shopee-api/query-product-detail) 接口及您的 `batch_id` 检查任务进度。

3. **获取数据 (Get Data)** ✅  
   任务完成后（`code = 0`），即可获取高保真、结构化的 Shopee 数据。
